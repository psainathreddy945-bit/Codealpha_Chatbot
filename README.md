# Artificial Intelligence Chatbot using Java


### Project Overview

This project is a Java-based Artificial Intelligence Chatbot developed as part of the CodeAlpha Internship Program. The chatbot interacts with users through a graphical user interface (GUI), processes user input using basic Natural Language Processing (NLP) techniques, and provides intelligent rule-based responses.

The chatbot is capable of answering frequently asked questions related to programming, Artificial Intelligence, Machine Learning, Data Science, courses, college information, date, time, and general conversations.

---

## Features

* Interactive chatbot communication
* User-friendly Java Swing GUI
* Real-time chat interface
* Basic NLP techniques

  * Lowercase conversion
  * Text preprocessing
  * Keyword matching
* Rule-based response generation
* Frequently Asked Questions (FAQ) support
* Date and Time responses
* Clear Chat functionality
* Exit functionality
* Object-Oriented Programming implementation

## How It Works

1. User enters a message in the text field.
2. The chatbot preprocesses the input.
3. NLP-based keyword matching is performed.
4. The chatbot searches for matching FAQs.
5. The appropriate response is generated.
6. The response is displayed in the chat window.

---

## NLP Techniques Used

### Text Preprocessing

* Converts text to lowercase
* Removes unnecessary spaces

Example:

Input:
What is JAVA?

Processed Input:
what is java

### Keyword Matching

The chatbot identifies keywords such as:

* java
* python
* ai
* machine learning
* college
* course

and returns the corresponding response.
